El DataSet ***salud_mundial.csv*** contiene indicadores de salud a nivel mundial e incluye información de 159 países, con datos como:
- país
- región
- nivel de ingresos
- esperanza de vida
- gasto en salud en dólares
- mortalidad infantil
- médicos por 1000 habitantes
- población urbana
- obesidad
- diabetes
- tabaquismo
- camas hospitalarias por 1000 habitantes
- vacunación

**Hallazgos:**
